== Description ==

Museo is a responsive WordPress theme that works great on business, organization or personal websites. Museums, churches, city halls, schools, kindergartens and other educational institutions, charities, governmental and nongovernmental organizations. The theme lets you choose the number of sidebars: on the left and right, only on the left, only on the right or no sidebars at all (full width). You can view the theme's demo here: https://demo.academiathemes.com/museo-lite/


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How to setup the front page like the demo site? =

The demo site URL:  https://demo.academiathemes.com/museo-lite/
Theme release page: https://www.academiathemes.com/free-wordpress-themes/museo-lite/

When you first activate Museo, your homepage will display posts in a traditional blog format. If you would like to add the additional featured sections from the demo, you can do that by going to Appearance > Customize > Theme Settings.

= Footer Area =

The footer area contains 3 widgetized columns of equal width.

== Quick Specs (all measurements in pixels) ==

	1. The main column width is 440.
	2. In full-width layout the content width is 960.
	3. The primary sidebar is 240 pixels wide and the secondary widget is 200 pixels wide.
	4. The featured image on the front page and on pages works best with images at least 300 wide.
	5. Featured Images for posts should be at least 300 wide.
	6. The footer widget columns are of equal width: 290.
